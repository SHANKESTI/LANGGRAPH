# 🧠 LangGraph Hands-On Implementations

This repository contains hands-on implementations and experiments using **LangGraph**, a framework for building **stateful, multi-agent workflows powered by Large Language Models (LLMs)**.

The project explores how LangGraph enables developers to build complex AI systems such as **AI agents, workflows, streaming pipelines, and tool integrations**.

---

# 🚀 Project Overview

LangGraph extends the capabilities of LangChain by enabling **graph-based orchestration of LLM applications**.  
It allows developers to create **stateful and multi-step reasoning workflows**.

This repository demonstrates practical implementations of various LangGraph components used in modern AI systems.

---

# 📂 Project Structure

### 🤖 AGENTS
Implementation of AI agents capable of performing tasks using tools and LLM reasoning.

### 👤 HITL (Human-In-The-Loop)
Workflows where humans interact with AI agents during execution.

### 🗄 LANGGRAPH_SQL
Examples of integrating SQL databases with LangGraph workflows.

### 📊 LANGSMITH
Experiments using **LangSmith** for monitoring and debugging AI workflows.

### 🧠 LLM
Integration of Large Language Models within LangGraph pipelines.

### 💾 PERSISTANCE
State persistence for long-running workflows and agents.

### 📦 PYDANTIC
Using Pydantic models for structured data validation.

### 🧩 STM (State Machine)
Implementation of state machines for managing agent workflows.

### ⚡ STREAMING
Streaming responses from LLMs in real time.

### 🔗 SUBGRAPH
Building modular graph components that can be reused.

### 🧰 TOOLS
Integration of external tools used by agents.

### 🔄 WORKFLOW
Multi-step workflows created using LangGraph.
